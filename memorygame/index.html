Instructions Game

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

What's fun about it?

During exercise memory games, set yourself a time each time you exercise, memory improves in its performance,

Why should you play?

Have you looked at your phone number and repeated it over and over to yourself, so that you can correctly dial it or pass it on to someone? This draws attention to the way you work and perform your memory. After a few moments of asking for the number correctly, chances of forgetting it are very likely. In order to remember things for a long time, the new information must be linked with the one you know, in order to install it with long-term memory, where information is stored more and for a period of time Longer period of time The solution is the memory game
